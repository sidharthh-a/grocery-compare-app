# Grocery Compare App

A quick commerce price comparison app that helps users find the cheapest grocery prices across platforms.

## Platforms
- Blinkit
- Zepto
- Swiggy Instamart

## Features
- Search grocery items
- Compare prices across platforms
- Highlight cheapest option
- Grocery list comparison (planned)
- Share savings (planned)

## Tech Stack
Backend: FastAPI (Python)  
Frontend: React / Next.js  
Database: PostgreSQL (planned)
